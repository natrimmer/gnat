This is first article using this new build system that I developed specifically for this site. I suspect this article should be initiated using: 

run init

Tongue in cheek aside, it has been fun and exciting developing this new site and build system. I've tried and dabbled with countless static site generators, but none of them have been quite right. I wanted something that had the development convience and extensibility of a modern framework yet produced something performant, simple, js free, and easy to maintain. I wanted a static site with as close to dynamic content as possible. Hence a build system that runs often on git pushes. The result should be a site that is easy to update and performs well on all devices. We'll see how it feels in the long run.
